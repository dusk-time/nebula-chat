## 📊 项目概览

**Nebula Chat** 是一个去中心化的 P2P 加密聊天应用，基于 Tauri 2.0 + React + libp2p 构建。

### 核心特性
- 🔐 **端到端加密** - Ed25519 身份系统
- 🌐 **去中心化** - 无服务器，纯 P2P 通信
- 🚀 **本地发现** - mDNS 自动发现局域网内的用户
- 💻 **跨平台** - Windows/macOS/Linux 支持

## 📝 使用流程

### 1. 首次启动
1. 输入用户名
2. 点击"生成新身份"
3. **备份身份信息**（JSON 格式）
4. 点击"我已备份，继续"

### 2. 添加联系人
1. 点击"+ 添加联系人"
2. 输入对方的 Peer ID
3. （可选）输入地址
4. 点击"添加"

### 3. 发送消息
1. 选择联系人
2. 输入消息
3. 按 Enter 或点击"发送"
